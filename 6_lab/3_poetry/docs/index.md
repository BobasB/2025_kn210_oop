# Лабораторна робота №6: Віртуальні середовища Python

## Огляд

Вітаємо у документації лабораторної роботи №6! Ця робота присвячена вивченню віртуальних середовищ Python та різних інструментів для роботи з ними.

## Мета роботи

Навчитися створювати та працювати з віртуальними середовищами Python, використовуючи різні інструменти управління залежностями.

## Що ви навчитеся

- 📦 Створювати ізольовані Python середовища
- 🔧 Працювати з **venv**, **Pipenv** та **Poetry**
- 📋 Керувати залежностями проекту
- 🗄️ Створювати Flask додатки з інтеграцією API
- 📚 Документувати проекти за допомогою MkDocs

## Демонстраційний проект

У цій роботі ми створимо Flask додаток для роботи з Jikan API (MyAnimeList), який:

- Отримує інформацію про аніме
- Зберігає дані у SQLite базі даних
- Надає веб-інтерфейс для перегляду

## Швидкий старт

```bash
# Клонуйте репозиторій
cd 6_lab/1_venv

# Створіть віртуальне середовище
python -m venv my_env

# Активуйте його
source my_env/bin/activate  # Linux/macOS
# або
my_env\Scripts\activate  # Windows

# Встановіть залежності
pip install -r requirements.txt

# Запустіть проект
python anime.py
```
